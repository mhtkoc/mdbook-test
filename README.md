This repository generates an mdBook from the contents of the `src` folder. All documentation files placed in `src` will be compiled into a navigable book format using mdBook.

## Usage

1. Place your markdown files in the `src` directory.
2. Run `mdbook build` to generate the book.
3. The output will be available in the `book` directory.

For more information, see the [mdBook documentation](https://rust-lang.github.io/mdBook/).
